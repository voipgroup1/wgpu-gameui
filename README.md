# wgpu-gameui

Custom wgpu-based game UI library originally extracted from a city builder game; widgets include button/checkbox/panel/slider/tabs/table/text_input/tooltip/progress_bar with a layout system and theme.

Import the crate as `wgpu_gameui`. The main API surface is re-exported from `src/lib.rs`, including widgets, `Theme`, layout primitives, `InputState`, `TextBlock`, and `TextRenderer`.
